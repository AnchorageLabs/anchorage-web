# Anchorage Labs Web

Private portfolio and landing page for Anchorage Labs.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Development

```bash
npm install
npm run dev
```

## Content Safety

The page uses public repository metadata and sanitized code excerpts. Secrets, private credentials, `.env` files, private keys, and internal-only implementation details must not be added to the site.
